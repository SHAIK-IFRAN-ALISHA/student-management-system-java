# Smart Student Management Portal

## Overview

A console-based Student Management System developed using Java.

The application allows administrators to manage student records through a menu-driven interface.

## Features

- Add Student
- View Student Records
- Search Student
- Delete Student
- Menu Driven Interface

## Technologies Used

- Java
- Object Oriented Programming (OOP)
- ArrayList
- Scanner Class

## Concepts Implemented

- Classes and Objects
- Constructors
- Encapsulation
- CRUD Operations
- Collections Framework

## Future Enhancements

- Admin Login
- Student Login
- Attendance Tracking
- Marks Management
- GPA Calculation
- Database Integration (MySQL)
